# 2022 개정 물리학: 빛과 물질 인터랙티브 학습 사이트

2022 개정 교육과정 물리학 3단원 '빛과 물질'의 모든 개념을 인터랙티브 시뮬레이션, 그래프, 도표를 통해 설명하는 교육용 웹사이트입니다.

## 📚 학습 주제

### 1. 빛의 파동성 (Wave Nature of Light)
- **개념**: 중첩과 간섭을 통한 빛의 파동 특성
- **인터랙티브 기능**:
  - 파동 중첩 시뮬레이션
  - 영의 이중슬릿 간섭 무늬 (파장, 슬릿 간격, 화면 거리 조절)
  - 간섭 강도 실시간 그래프
- **응용**: 회절 격자, 홀로그래피, 광학 코팅, 간섭계

### 2. 굴절과 렌즈 (Refraction and Lenses)
- **개념**: 빛의 굴절 현상과 렌즈에서의 상 맺힘
- **인터랙티브 기능**:
  - 렌즈 시뮬레이션 (물체 거리, 초점거리, 굴절률 조절)
  - 렌즈 공식 자동 계산 (1/f = 1/o + 1/i)
  - 상의 성질 판정 (실상/허상, 배율)
  - 광선 추적 다이어그램
  - 스넬의 법칙 그래프
- **응용**: 안경, 카메라, 현미경, 포토리소그래피

### 3. 빛과 물질의 이중성 (Duality of Light and Matter)
- **개념**: 광전 효과와 물질파를 통한 이중성 이해
- **인터랙티브 기능**:
  - 광전 효과 시뮬레이션 (진동수, 일함수, 파장 조절)
  - 최대 운동에너지 계산
  - 드 브로이 파장 계산
  - 파장 vs 광전 효과 그래프
- **응용**: 태양전지, 전자 현미경, 영상 정보 저장, 광전자 증배관

### 4. 원자 구조와 에너지 준위 (Atomic Structure and Energy Levels)
- **개념**: 보어의 원자 모형과 양자화된 에너지 준위
- **인터랙티브 기능**:
  - 보어의 원자 모형 시각화
  - 에너지 준위 다이어그램
  - 수소 원자 선스펙트럼 (발머 계열)
  - 여기와 이완 과정 설명
- **응용**: 형광등, LED, 분광 분석, 레이저, 형광 화면

### 5. 에너지띠와 반도체 (Energy Band and Semiconductors)
- **개념**: 고체의 에너지띠 구조와 반도체 소자의 원리
- **인터랙티브 기능**:
  - 도체, 반도체, 절연체의 에너지띠 구조 비교
  - 온도에 따른 전도도 변화 그래프
  - p-n 접합 구조 및 다이오드 원리
  - 다이오드 I-V 특성 곡선
- **응용**: LED, 정류 다이오드, 태양전지, 트랜지스터

### 6. 특수 상대성 이론 (Special Relativity)
- **개념**: 광속 불변과 시간 팽창, 길이 수축
- **인터랙티브 기능**:
  - 시간 팽창과 길이 수축 시뮬레이션
  - 로렌츠 인수 계산
  - 상대론적 에너지 그래프
  - 뮤온 수명 예제
- **응용**: GPS 시스템, 핵 에너지, 입자 가속기, 우주론

## 📦 설치 및 실행

### 요구사항
- Node.js 22.13.0 이상
- pnpm 11.7.0 이상

### 설치
```bash
cd physics-light-and-matter
pnpm install
```

### 개발 서버 실행
```bash
pnpm dev
```

개발 서버는 `http://localhost:5173`에서 실행됩니다.

### 프로덕션 빌드
```bash
pnpm build
```

### 빌드 결과 미리보기
```bash
pnpm preview
```
